İçerikmatik İçeri Aktarma Aracı ile İçerikmatik API servisini kullanarak içeriklerinizi Wordpress sitenize aktarabilirsiniz.

== Description ==

İçerikmatik API servisini kullanarak içeriklerinizi Wordpress sitenize aktarabilirsiniz.


== Installation ==

1. Eklenti dosyalarını `/wp-content/plugins/icerikmatik-import` dizinine yükleyin ya da WordPress eklenti sayfasından eklentinizi yükleyebilirsiniz.
1. Wordpress sitenizdeki Eklentiler sayfasından eklentiyi etkinleştirin.
1. İçerikmatik hesabınızla erişim anahtarınızı oluşturun.
1. Menüden İçerikmatik > Ayarlar sayfasına giderek erişim anahtarınızı Wordpress websitenize ekleyin.
